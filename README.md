# LaviLog

A website that tracks a baby's food tastes.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v20+)

### Installation

```bash
npm install
```

### Running the App

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
npm run preview
```

## Usage

1. **Add a food** - Fill in the food name, quantity, and how many times the baby ate it, then click "Add Food"
2. **Edit a food** - Click "Edit" on any food card to update its details
3. **Delete a food** - Click "Delete" to remove a food entry

All data is saved locally in your browser, so it persists between sessions.
